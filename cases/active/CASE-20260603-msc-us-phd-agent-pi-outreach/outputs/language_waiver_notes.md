# 语言考试与豁免说明（非法律意见，申请前须逐校核对）

## 「入学前语言尽量考掉」是什么意思？

指在 **2026-09 赴曼大前**（或最迟 **2026-12 美国 PhD 网申截止前**）拿到 **TOEFL 或 IELTS** 的官方成绩，并达到目标校最低线（常见：TOEFL iBT 90–100+，IELTS 6.5–7.0+，**各系不同**）。

原因：

- 美国 PhD **网申系统**通常要填语言分数；  
- 推荐信、SOP 可后补，但 **语言分经常卡提交**；  
- 你选 **策略 A（2027 Fall）** → 申请窗约在 **2026-12 ~ 2027-01**，与 MSc 第 1 学期重叠，**不能拖到毕业后再考**（除非改策略 B）。

## 能否不考？和「UoM 学历豁免」的关系

**没有全国统一豁免**；**一所学校一条政策**。

| 情况 | 常见结果 |
|------|----------|
| 仅安大本科（中文授课为主） | **多数美国学校仍要** TOEFL/IELTS |
| 已完成英国/英语国家学位 | 许多学校 **可豁免**（常要求满 2 年或整学位） |
| **在读** 曼大 MSc，申请时仅 3–4 个月 | **多数学校仍要成绩**；部分接受「在读英语授课项目」证明 — **少数**，须邮件问 grad admission |
| **2027 毕业后** 持曼大学位再申（策略 B） | **更多学校**可能豁免（凭 UK MSc 英语授课） |

**结论（本案策略 A）：**

- **不要默认**「有 UoM offer / 在读就能免考」。  
- **务实做法：**  
  1. **2026 年夏（入学前）** 考一次 **IELTS 或 TOEFL**（UK 申学常考 IELTS UKVI/Academic；美国 PhD 两者多可接受，看校规）。  
  2. 并行做 **豁免 inquiry**：对目标 15–20 校发邮件问：*Currently enrolled in English-taught MSc at UoM since Sept 2026; TOEFL/IELTS waiver if degree not yet completed?*  
  3. 若某校明确书面豁免，再 **不考** 该校；其余校仍用成绩。

## 和曼大（UoM）本身的关系

- UoM 录取对你的语言要求 **≠** 美国 PhD 对你的语言要求。  
- 美国看的是 **你能不能在美国用英语做研究/教课（TA）**。

## 最低行动（不授权代考/代申）

- [ ] 选定考试：IELTS Academic **或** TOEFL iBT（按你目标校列表多数偏好）。  
- [ ] 目标分：先按 **TOEFL 100 / IELTS 7.0** 备考（可下调到 match 校）。  
- [ ] 考试窗口建议：**2026-06 ~ 2026-08**（入学前），留 1 次重考余地。  
- [ ] 建表：`outputs/target_schools_language.csv`（校名、是否豁免、最低分、deadline）— 可后续 Phase 再填。
